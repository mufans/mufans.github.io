Hexo指令

2022-02-17

hexo提供了一些指令，方便我们快速搭建博客和发布文章

  1. 初始化项目目
  • folder 为可选参数表示初始化目录，默认为当前目录
  1. 新建文章
  • layout 有三种布局: post (直接发布的文章), draft (文章草稿) , page** (新的文章), 分别放在不同路径。 默认为post类型。
    layout path
    draft source/_drafts
    post source/_posts
    page source
  1. page

执行完会在source下创建newPage目录和index.md,之后就可以访问 https://xx.xx/newPage 主页下的内容了。

  1. draft

draft 是文章的草稿，文章在发布前可以先创建草稿

  • hexo g 简写
